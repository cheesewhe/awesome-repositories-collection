## Файловая навигация

Основные инструменты для эффективного изучения и навигации по файловой системе.

- **[fzf](https://github.com/junegunn/fzf)** (★ 75k) — командно-линейный нечеткий искатель с интерактивным интерфейсом. Интегрируется с историей оболочки, поиском файлов и vim/neovim для быстрой навигации.
- **[bat](https://github.com/sharkdp/bat)** (★ 55.8k) — клон кошки с подсветкой синтаксиса, интеграцией git и автоматической подачей. Идеально подходит для быстрого просмотра файлов кода в терминале.
- **[fd](https://github.com/sharkdp/fd)** (★ 40.5k) — простая, быстрая и удобная для пользователя альтернатива `find`. Поддерживает параллельное выполнение, игнорирует шаблоны и интеллектуальную чувствительность.
- **[Flameshot](https://github.com/flameshot-org/flameshot)** (★ 28.2k) — мощный инструмент для скриншотов с возможностью аннотации. Захват, аннотирование и обмен скриншотами со встроенным редактором изображений.
- **[exa](https://github.com/ogham/exa)** (★ 24.2k) — современная замена `ls` с интеграцией git, видом на дерево и цветными типами файлов для повышенной читаемости.
- **[ranger](https://github.com/ranger/ranger)** (★ 16.6k) — Ви-вдохновленный файловый менеджер с макетом из трех колонок, предварительным просмотром файлов и широкими возможностями настройки.
- **[lsd](https://github.com/lsd-rs/lsd)** (★ 15.1k) — XE`ls` следующего поколения F команда с иконками, цветами и видом на дерево. Написано в ржавчине для пылающего исполнения.
- **[CopyQ](https://github.com/hluk/CopyQ)** (10.7k) — Расширенный менеджер буфера обмена с историей поиска. Храните и организуйте записи в буфере обмена с тегами, заметками и поддержкой сценариев.
- **[qView](https://github.com/jurplel/qView)** (2.9k) — минималистичный и быстрый просмотрщик изображений для настольных компьютеров. Легкий с клавиатурной навигацией, поддерживает все основные форматы изображений.
- **[cloudcmd](https://github.com/coderaiser/cloudcmd)** (★ 2k) — веб-файловый менеджер с интегрированной консолью и редактором. Доступ и удаленное управление файлами через интерфейс браузера.
