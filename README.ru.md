## Навигация по файлам

Основные инструменты для эффективного изучения и навигации по файловой системе.

**[fzf](https://github.com/junegunn/fzf)** — командный нечеткий искатель с интерактивным интерфейсом. Интегрируется с историей оболочки, поиском файлов и vim/neovim для быстрой навигации.
[fd](https://github.com/sharkdp/fd) - простая, быстрая и удобная альтернатива `find`. Поддерживает параллельное выполнение, игнорирует шаблоны и интеллектуальную чувствительность.
**[exa](https://github.com/ogham/exa)** — современная замена `ls` с интеграцией git, видом на дерево и цветными типами файлов для повышенной читаемости.
**[bat](https://github.com/sharkdp/bat)** — клон кошки с подсветкой синтаксиса, интеграцией git и автоматической подачей. Идеально подходит для быстрого просмотра файлов кода в терминале.
**[lsd](https://github.com/lsd-rs/lsd)** - XE`ls` следующего поколения F команда с иконками, цветами и видом на дерево. Написано в ржавчине для пылающего исполнения.
**[ranger](https://github.com/ranger/ranger)** — Ви-вдохновленный файловый менеджер с макетом из трех столбцов, предварительным просмотром файлов и широкими возможностями настройки.
[qView](https://github.com/jurplel/qView) — минималистичный и быстрый просмотрщик изображений для настольных компьютеров. Легкий с клавиатурной навигацией, поддерживает все основные форматы изображений.
**[cloudcmd](https://github.com/coderaiser/cloudcmd)** — веб-файловый менеджер с интегрированной консолью и редактором. Доступ и удаленное управление файлами через интерфейс браузера.
**[Flameshot](https://github.com/flameshot-org/flameshot)** — мощный инструмент для скриншотов с возможностью аннотации. Захват, аннотирование и обмен скриншотами со встроенным редактором изображений.
**[CopyQ](https://github.com/hluk/CopyQ)** — Расширенный менеджер буфера обмена с историей поиска. Храните и организуйте записи в буфере обмена с тегами, заметками и поддержкой сценариев.

### Поиск и замена

Мощные инструменты для поиска по кодовым базам и выполнения объемных текстовых операций.

**[ripgrep](https://github.com/BurntSushi/ripgrep)** — Ультрабыстрый рекурсивный инструмент поиска, который по умолчанию уважает .gitignore. Превосходит грэп, аг и другие альтернативы на больших кодовых базах.
**[ag (The Silver Searcher)](https://github.com/ggreer/the_silver_searcher)** — инструмент для поиска кода, оптимизированный для разработчиков. Быстрее, чем ack, с умными по умолчанию для игнорирования каталогов VCS.
**[sd](https://github.com/chmln/sd)** — Интуитивный инструмент поиска и замены CLI с поддержкой регекса. Безопаснее и эргономичнее, чем `sed` для повседневного использования.
**[ast-grep](https://github.com/ast-grep/ast-grep)** — инструмент поиска и рефакторинга структурного кода. Поисковый код по шаблонам AST вместо регекса для более точных результатов.

##Системный мониторинг

Отслеживание системных ресурсов, процессов и показателей производительности в режиме реального времени.

**[htop](https://github.com/htop-dev/htop)** — интерактивный просмотрщик процессов для систем Unix. Цветной дисплей с поддержкой мыши и настраиваемыми колонками.
**[btop](https://github.com/aristocratos/btop)** — монитор ресурсов с красивым пользовательским интерфейсом, показывающий процессор, память, диск, сеть и информацию об обработке. Современная реализация C++ с темами.
**[glances](https://github.com/nicolargo/glances)** — кроссплатформенный инструмент мониторинга, написанный на Python. Экспорт данных в различные форматы и поддержка клиент-серверного режима.
**[ncdu](https://dev.yorhel.nl/ncdu)** — анализатор использования дисков на основе NCurses. Быстро найдите то, что занимает дисковое пространство с интуитивно понятным интерфейсом.
**[bottom](https://github.com/ClementTsang/bottom)** - Графический процесс/системный монитор, вдохновленный gtop и gotop. Настраиваемые виджеты с кроссплатформенной поддержкой.
- **[ctop](https://github.com/bcicen/ctop)** - Топ-подобный интерфейс для контейнерных метрик. Мониторинг контейнеров Docker в режиме реального времени с помощью статистики использования ресурсов.
**[Performa](https://github.com/jhuckaby/Performa)** — мониторинг сервера с использованием пользовательских метрик. Отслеживание производительности в режиме реального времени с настраиваемыми оповещениями и приборными панелями.
**[resources](https://github.com/nokyan/resources)** — монитор системных ресурсов для CPU, GPU и NPU. Легкий инструмент, показывающий подробное использование оборудования.
**[Umami](https://github.com/umami-software/umami)** — веб-аналитика, ориентированная на конфиденциальность. Самостоятельно размещенная альтернатива Google Analytics с соблюдением GDPR.
**[Healthchecks](https://github.com/healthchecks/healthchecks)** — служба мониторинга рабочих мест Cron. Получайте оповещения, когда запланированные задачи не выполняются или не выполняются вовремя.
**[coroot](https://github.com/coroot/coroot)** - Мониторинг инфраструктуры и анализ APM. Выявить узкие места производительности и оптимизировать доставку приложений.
**[Netdata](https://github.com/netdata/netdata)** — мониторинг производительности систем и приложений в режиме реального времени. Распределенный, в режиме реального времени, мониторинг состояния здоровья и устранение неполадок.
[VictoriaMetrics](https://github.com/VictoriaMetrics/VictoriaMetrics) - Быстрое, экономичное решение для мониторинга и база данных временных рядов. Долгосрочное удаленное хранилище для Prometheus с высокой производительностью и масштабируемостью.
**[Signoz](https://github.com/SigNoz/signoz)** — APM с открытым исходным кодом и платформа наблюдения. Наблюдение с полным стеком с бревнами, метриками и следами в одной панели из стекла.
**[Uptrace](https://github.com/uptrace/uptrace)** - Распределенный инструмент отслеживания и APM. APM с открытым исходным кодом и распределенное решение для отслеживания, совместимое с OpenTelemetry.
**[Sentry](https://github.com/getsentry/sentry)** — мониторинг приложений и отслеживание ошибок. Отслеживание ошибок с открытым исходным кодом, которое помогает разработчикам отслеживать и исправлять сбои в режиме реального времени.
**[Grafana Loki](https://github.com/grafana/loki)** — система агрегации журналов, вдохновленная Прометеем. Высокоэффективное хранение журналов и запрос с интеграцией Grafana для централизованной регистрации.
**[PowerToys](https://github.com/microsoft/PowerToys)** — утилиты Windows для пользователей. Коллекция инструментов для расширения функциональности Windows с помощью ярлыков, выбора цвета и многого другого.
**[starship](https://github.com/starship/starship)** — минималистичная поперечная оболочка. Быстрая и настраиваемая подсказка для любой оболочки со статусом git, заданиями и информацией каталога.
**[Quick Look](https://github.com/QL-Win/QuickLook)** — мгновенный предварительный просмотр файлов для Windows. Нажмите пробел для предварительного просмотра файлов без открытия приложений.
**[Atlas OS](https://github.com/AtlasOS/Atlas)** — легкая оптимизация Windows с открытым исходным кодом. Настраиваемый дистрибутив Windows ориентирован на производительность и минимализм.
**[Home Assistant](https://github.com/home-assistant/core)** — платформа домашней автоматизации с открытым исходным кодом. Умный домашний хаб с управлением устройствами, автоматизацией и обширной интеграционной экосистемой.
**[Qubes OS](https://github.com/QubesOS/qubes-os)** — настольная операционная система, ориентированная на безопасность. Xen OS использует виртуализацию для изоляции различных частей системы для повышения безопасности.

**[NixOS](https://github.com/NixOS/nixpkgs)** — Декларативный дистрибутив Linux с воспроизводимыми сборками. Чисто функциональный менеджер пакетов с атомными обновлениями и откатами.

###Сеть

Инструменты для тестирования API, отладки сетевого трафика и управления соединениями.

**[httpie](https://github.com/httpie/cli)** — удобный для пользователя HTTP-клиент для тестирования API. Экспрессивный синтаксис с выделением синтаксиса и поддержкой JSON из коробки.
**[curlie](https://github.com/rs/curlie)** — современный кулон с httpie-подобным интерфейсом. Объединяет силу завитка с удобным синтаксисом httpie.
**[dog](https://github.com/ogham/dog)** — современный DNS-клиент с цветным выходом и поддержкой DNS-over-HTTPS. Лучшая альтернатива `dig` с более четким выходом.
**[mitmproxy](https://github.com/mitmproxy/mitmproxy)** — интерактивный HTTPS-прокси для тестировщиков и разработчиков. Проверяйте, модифицируйте и воспроизводите трафик HTTP/HTTPS.
**[ngrok](https://github.com/inconshreveable/ngrok)** — обратный прокси-сервер для создания безопасных туннелей к локальному хосту. Необходим для тестирования веб-хук и раскрытия местных услуг.
**[bandwhich](https://github.com/imsnif/bandwhich)** - Инструмент использования полосы пропускания терминала. Показывает текущее использование сети по процессу, подключению и удаленному IP.
**[graftcp](https://github.com/hmgle/graftcp)** — Прозрачный TCP-прокси для любого приложения. Перенаправить TCP соединения без изменения кода приложения или конфигурации.
**[easy-postman](https://github.com/lakernote/easy-postman)** — Инструмент для тестирования нагрузки и интеграции API. Упрощенная альтернатива Postman с возможностями автоматизированного тестирования.
**[share](https://github.com/schollz/share)** - Сквозная передача зашифрованных файлов через Интернет или CLI. Безопасный одноранговый обмен файлами без промежуточных серверов.
**[FileZilla](https://filezilla-project.org/)** — легкий FTP-клиент для передачи файлов. Загружайте файлы на серверы и редактируйте код непосредственно на удаленных серверах.
**[RustDesk](https://github.com/rustdesk/rustdesk)** — программное обеспечение для удаленного рабочего стола с открытым исходным кодом. Автономная альтернатива AnyDesk для удаленного доступа и поддержки.
**[LocalSend](https://github.com/localsend/localsend)** — безопасный обмен файлами по локальной сети. Зашифрованная передача файлов между устройствами без облака или интернета.
**[Bruno](https://github.com/usebruno/bruno)** — API-клиент для тестирования REST, GraphQL и SOAP API. Легкая альтернатива Postman и Insomnia с автономной архитектурой.
[NETworkManager](https://github.com/BornToBeRoot/NETworkManager) - Сетевой инструментарий с анализатором WiFi, сканером портов и управлением RDP/SSH. Профессиональный инструмент сетевого администрирования для Windows.
**[Wireshark](https://freecodecamp.org/news/use-wireshark-filters-to-analyze-network-traffic)** — анализатор сетевых протоколов для отладки трафика. Инструмент с открытым исходным кодом для устранения неполадок в сети, анализа и разработки протоколов.
**[Tailscale VPN](https://freecodecamp.org/news/set-up-a-home-vpn-on-a-raspberry-pi)** — Mesh VPN с бесплатным клиентом с открытым исходным кодом и самостоятельным хостингом. Zero-config VPN для создания защищенных сетей между устройствами.
**[Pi-hole](https://github.com/pi-hole/pi-hole)** — блокировка рекламы по всей сети с помощью собственного оборудования Linux. Самостоятельная воронка DNS, которая блокирует рекламу и трекеры на сетевом уровне.
**[Postman](https://freecodecamp.org/news/master-api-testing-with-postman)** — бесплатная версия Postman для тестирования API. Полная среда разработки API с возможностями автоматизированного тестирования.
**[Mirotalk](https://github.com/mirotalk/mirotalk)** — P2P видеоконференция через WebRTC. Простая и быстрая альтернатива Zoom и Google Meet с одноранговыми соединениями.
**[Chatwoot](https://github.com/chatwoot/chatwoot)** — платформа взаимодействия с клиентами с открытым исходным кодом. Унифицированный почтовый ящик для всех разговоров клиентов по нескольким каналам.

**[delta](https://github.com/dandavison/delta)** — пейджер с синтаксическим подсветкой для вывода git, diff и grep. Делает обзор кода более приятным с побочными диффами.

**[FFmpeg](https://github.com/FFmpeg/FFmpeg)** — Полное кроссплатформенное решение для записи, преобразования и потоковой передачи аудио и видео. Отраслевой стандарт для мультимедийной обработки.
**[yt-dlp](https://github.com/yt-dlp/yt-dlp)** — многофункциональный видеозагрузчик командной строки. Вилка youtube-dl с дополнительными функциями и исправлениями.
**[Sora Extend](https://github.com/mshumer/sora-extend)** - Инструмент для цепной передачи и расширения видеопоколения OpenAI Sora 2 за пределы 12 секунд. Автоматизированная быстрая деконструкция и сегментная конкатенация.
**[HandBrake](https://github.com/HandBrake/HandBrake)** — видеотранскодер с поддержкой полного формата. Версии GUI и CLI для пакетной обработки.
**[Kdenlive](https://github.com/KDE/kdenlive)** — бесплатный видеоредактор с открытым исходным кодом. Профессиональное нелинейное программное обеспечение для редактирования видео с многодорожечным редактированием и эффектами.
**[wunjo](https://github.com/wladradchenko/wunjo)** — генератор анимации, управляемый движением головы из видео. Инструмент глубокого обучения для создания реалистичной анимации лица.
**[auto-subs](https://github.com/tmoroney/auto-subs)** — автоматический генератор субтитров для видео. Offline-совместимый инструмент, использующий распознавание речи для создания файлов субтитров.
**[shutter-encoder](https://github.com/paulpacifico/shutter-encoder)** — усовершенствованный видеокодер и оптимизатор. Конверсия видео профессионального уровня с пакетной обработкой и оптимизацией формата.
**[Jellyfin](https://github.com/jellyfin/jellyfin)** — самохостинг медиасервера. Бесплатная и открытая альтернатива Plex и Emby для потоковой передачи вашей медиа коллекции.
**[LibrePhotos](https://github.com/LibrePhotos/librephotos)** — Служба самостоятельного управления фотографиями. Альтернатива Google Photos с открытым исходным кодом с распознаванием лиц и автоматической меткой.
**[Upscayl](https://github.com/upscayl/upscayl)** — инструмент для масштабирования изображений на основе ИИ. Улучшение качества изображения с использованием моделей машинного обучения на местном уровне.
[Shotcut](https://www.shotcut.org/) - мощный и простой видеоредактор с регулярными обновлениями. Идеально подходит для учебного контента и базового редактирования видео / аудио.
**[OBS Studio](https://obsproject.com/)** — программное обеспечение для потоковой передачи и записи с открытым исходным кодом. Запись рабочего стола, потоковая передача на Twitch/YouTube и захват вызовов для архивов.
**[Blender](https://www.blender.org/)** — профессиональный набор для создания 3D. Моделирование, анимация, рендеринг и композитинг для фильмов, игр и визуальных эффектов.
**[Audacity](https://www.audacityteam.org/)** — бесплатный аудиоредактор с открытым исходным кодом. Запись, редактирование и смешивание аудиодорожек с инструментами профессионального качества.
[VLC Media Player](https://www.videolan.org/vlc/) — универсальный медиаплеер. Он воспроизводит практически все видео и аудио форматы без установки кодека.
[MPV](https://mpv.io/) — легкий, мощный медиаплеер. Командная строка, основанная на минимальном графическом интерфейсе, настраиваемая и сценарная.
**[Immich](https://github.com/immich-app/immich)** — решение для резервного копирования фотографий и видео. Альтернатива Google Photos с автоматическим резервным копированием и распознаванием лиц.
[Pars Local Player (PLP)](https://github.com/pars-local-player/pars-local-player) - Легкий видеоплеер без телеметрии или отслеживания. Медиаплеер, ориентированный на конфиденциальность, с чистым интерфейсом.
**[Sora 2 API Video Generator](https://github.com/sora-ai/video-generator)** — видеогенератор с открытым исходным кодом с API. Создание видео с использованием передовых моделей ИИ и методов генерации.
**[Vexa](https://github.com/Vexa-ai/vexa)** — Самостоятельный API для автоматизации транскрипции. Решение с открытым исходным кодом для автоматического преобразования аудио в текст и резюме встреч.
**[Audiobook Generator](https://github.com/BookxDev/bookxAI)** — создание аудиокниги на основе ИИ из текста. Инструмент с открытым исходным кодом, который генерирует естественно звучащие аудиокниги с использованием технологии преобразования текста в речь.
**[Eclipsa Audio](https://goo.gle/41j1MRl)** — бесплатный формат и инструменты 3D-аудио с открытым исходным кодом. Пространственная аудиотехнология Google для погружения в звук.
**[Godot Engine](https://github.com/godotengine/godot)** — бесплатный и открытый игровой движок 2D и 3D. Кроссплатформенная разработка игр с визуальным сценарием и обширным набором функций.
**[Veloren](https://github.com/veloren/veloren)** — многопользовательская ролевая игра с открытым исходным кодом. Общественная игра с процедурным мировым поколением и совместным геймплеем.
**[OpenTTD](https://github.com/OpenTTD/OpenTTD)** — игра-симулятор транспортного бизнеса с открытым исходным кодом. Усовершенствованная версия Transport Tycoon Deluxe с многопользовательским режимом онлайн.
**[SuperTuxKart](https://github.com/supertuxkart/stk-code)** — Бесплатная 3D-картовая гоночная игра. Веселая гоночная игра с участием Tux и друзей с различными треками и режимами.

3D Vision & Scanning

Инструменты и библиотеки с открытым исходным кодом для 3D компьютерного зрения, сканирования, реконструкции и обработки облаков точек.

**[OpenScan](https://openscan.eu)** — модульный 3D-сканер с открытым исходным кодом с фотограмметрией. Включает в себя схемы, документацию и программное обеспечение для сборки DIY. Процесс сканирования локально или в облаке для 3D-печати, реверс-инжиниринга и цифровой сохранности.
**[Meshroom](https://github.com/alicevision/meshroom)** — Мощный фотограмметрический трубопровод с открытым исходным кодом для 3D-реконструкции. Узловой интерфейс визуального программирования, полная обработка от фотографий до 3D-моделей, Python API. Используется в науке, археологии и разработке игр.
**[PiLiDAR](https://github.com/iLiAR/PLiDAR)** — проект 3D-сканера DIY LiDAR с использованием Raspberry Pi и камеры. Открытое аппаратное и программное обеспечение с лицензией CC-NC-SA. Дешевый лазерный сканер для экспериментов с облаком точек.
**[Open3D](https://github.com/isl-org/Open3D)** — Всеобъемлющая библиотека для обработки 3D-данных. Точечные облачные манипуляции, генерация сеток, визуализация, регистрация сканирования. Поддержка Python и C++ с обширной документацией.
**[CloudCompare](https://github.com/CloudCompare/CloudCompare)** — инструмент обработки и анализа облаков с открытым исходным кодом. Стандартные форматы импорта / экспорта, генерация сеток, фильтрация и скриптинг. Необходим для сравнения, слияния и анализа облаков точек.
**[COLMAP](https://github.com/colmap/colmap)** - Индустриально-стандартный инструмент Структура-из-Движения и фотограмметрия. Реконструируйте 3D-сцены из нескольких изображений с автоматической калибровкой камеры и плотными конвейерами реконструкции.
[OpenCV](https://github.com/opencv/opencv) — фундаментальная библиотека компьютерного зрения. Включает алгоритмы для стереовидения, обработки глубины, генерации облака точек, стереокамер и постобработки.
**[Salingo Virtual 3D Scanner](https://github.com/Salingo/virtual-3d-scanner)** — синтетический RGB-D-образ и точечный облачный генератор. Сканирование виртуальных 3D-моделей для создания наборов данных для обучения и развития компьютерного зрения.
**[TripoSR](https://github.com/VAST-AI-Research/TripoSR)** — инструментарий глубокого обучения для однофокусной 3D-реконструкции. Реконструкция поверхности на основе ИИ с быстрым выводом и высокой точностью для современных алгоритмов ИИ. [Website](https://triposrai.com/)
**[Potree](https://github.com/potree/potree)** — облачный рендеринг на основе WebGL для больших наборов данных. Интерактивный браузер, поддерживающий миллионы точек с помощью инструментов рендеринга и измерения уровня детализации.
**[PointLLM](https://github.com/OpenRobotLab/PointLLM)** - Расширяет большие языковые модели для понимания облаков точек. Обеспечивает нулевое 3D-обсуждение, ответы на вопросы и понимание сцены из данных облака точек. [ECCV 2024 Лучший кандидат на работу]
**[PCL (Point Cloud Library)](https://github.com/PointCloudLibrary/pcl)** — Полная библиотека для обработки 2D/3D изображений и облаков точек. Отраслевой стандартный набор инструментов для фильтрации, сегментации, регистрации, реконструкции поверхности и оценки характеристик.
[ReBound](https://github.com/ramdrop/ReBound) - инструмент с открытым исходным кодом для визуализации и аннотирования данных LiDAR. Предназначен для активных систем обучения в автономных транспортных средствах с интуитивно понятным интерфейсом 3D-аннотации.
[pyRANSAC-3D](https://github.com/leomariga/pyRANSAC-3D) — инструмент Python для установки примитивных 3D-форм в облаках точек с использованием алгоритма RANSAC. Быстрое и надежное геометрическое примитивное обнаружение (планы, сферы, цилиндры и т.д.).

**[Rapid SCADA](https://github.com/RapidScada/Scada)** — русскоязычный проект SCADA с поддержкой Siemens S7, Allen-Bradley, Arduino, Raspberry Pi. Гибкая визуализация, отчетность, интеграция с промышленными сетями.
[OpenAPC](http://www.openapc.com) - платформа с открытым исходным кодом для промышленного контроля и визуализации. Расширяется с помощью пользовательских плагинов для специализированных приложений.
**[OpenPLC](https://www.openplcproject.com)** | [GitHub](https://github.com/thiagoralves/OpenPLC_v3) — Комплексная платформа с открытым исходным кодом для программирования ПЛК (IEC 61131-3: Ladder, FBD, ST, IL, SFC). Моделирование, развертывание в Arduino/Raspberry Pi и промышленное оборудование, веб-мониторинг, Modbus TCP/RTU, интеграция SCADA.
[Beremiz IDE](https://github.com/beremiz/beremiz) — платформа PLC с интеграцией Python. Отлично подходит для сложных распределенных проектов с возможностями разработки пользовательских драйверов.
**[PLC Fiddle](https://www.plcfiddle.com)** — веб-симулятор ПЛК (IEC 61131). Изучайте и отлаживайте программы PLC без установки программного обеспечения.
[Pigweed SDK](https://goo.gle/4fA1coO) - Инструменты для разработки встраиваемых систем. Встроенная платформа разработки Google с открытым исходным кодом с драйверами устройств, библиотеками и инструментами производительности.
[TensorFlow Smart Buildings Simulator](https://goo.gle/oTOwjRBPmo) — открытый симулятор для управления энергопотреблением зданий. Платформа Google с открытым исходным кодом для разработки и тестирования алгоритмов управления интеллектуальным зданием.
